APNDE Framework

Autonomous Polymer Nanocomposite Discovery Engine — Architecture, Data Flow & Methodology

System Architecture

Pipeline Data Flow

1
PDF Ingestion — PyMuPDF opens each PDF in ./pdf, extracts full text (up to 120k chars) and all embedded images (filtering out icons <150px).
2
Miner Extraction — Full text is sent to GPT-4o with a 6k-char structured prompt. The LLM returns a JSON with 29 named fields per sample covering composition, chain conformation (Rg, Rg/Rg₀), scattering parameters (ξ, Porod exponent), surface properties (σ, brush height), and thermal/mechanical data.
3
Scattering Plot Identification — Each extracted image is sent to GPT-4o Vision to classify: is it a SANS/SAXS/S(q)/P(q)/Kratky plot? Returns plot type, axis labels, units, and confidence level.
4
Plot Digitization — Identified scattering plots are sent back to GPT-4o Vision with instructions to extract numerical q and I(q) pairs. Data is normalized to q in Å⁻¹ and I(q) in cm⁻¹. Multi-curve plots produce multiple labeled datasets. Log-log PNG plots are generated from the extracted data using matplotlib.
5
Neutron Contrast & Simulation — For each sample, the SLD lookup table maps materials to scattering length densities. (Δρ)² is computed. SasModels runs sphere or cylinder models with the extracted parameters (radius, volume fraction, SLD values) to produce simulated I(q) curves over q = 0.001–0.5 Å⁻¹.
6
Dashboard Generation — All results are aggregated into an interactive HTML dashboard with clickable stat cards, per-paper sample tables (with Rg, conformation badges, property tags), experimental scattering plot galleries, and a modal lightbox.

Neutron Scattering Contrast

Small-angle neutron scattering (SANS) measures structural features at the 1–100 nm scale. The scattered intensity depends on the contrast between components:

I(q) ∝ (Δρ)² · φ · V · P(q) · S(q)

where Δρ = ρfiller − ρmatrix is the scattering length density difference, φ is the volume fraction, V is the particle volume, P(q) is the form factor, and S(q) is the structure factor.

SasModels Simulation

The simulator auto-selects a scattering model based on filler geometry:

Sphere Model

Used for spherical nanoparticles (SiO₂, Au, PS NPs). Parameters: radius (Å), SLD_particle, SLD_solvent, scale (volume fraction), background.

P(q) = [3(sin(qR) − qR·cos(qR)) / (qR)³]²

Cylinder Model

Used for carbon nanotubes (SWCNT, MWCNT). Parameters: radius (Å), length (Å), SLD values, scale, background.

P(q) = ∫₀^{π/2} f²(q,α) sin(α) dα

where f(q,α) combines Bessel J₁ and sinc functions over orientation angle α

Key Chain Conformation Parameters

A central goal of PNC SANS studies is measuring how nanoparticle fillers perturb polymer chain conformations. The pipeline extracts:

ParameterSymbolUnitSignificance
Radius of gyrationRgÅRMS distance of chain segments from center of mass. Measured via Guinier or Debye fits.
Bulk RgRg₀ÅRg of the neat polymer (no filler) as reference baseline.
Rg ratioRg/Rg₀—>1 = chain expansion; <1 = chain contraction; =1 = unperturbed.
Conformation state——Qualitative label: expanded, contracted, or unchanged relative to bulk.
Correlation lengthξÅCharacteristic length scale of concentration fluctuations (Ornstein-Zernike fits).
Porod exponentn—High-q power law slope. n=4 (smooth surface), n=3 (rough), n=2 (Gaussian chain).
